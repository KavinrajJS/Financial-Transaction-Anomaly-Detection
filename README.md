# Financial-Transaction-Anomaly-Detection
## 📌 Project Overview
This project detects fraudulent financial transactions using Machine Learning algorithms and visualizes the results using Microsoft Power BI.

## 🎯 Objective
- Detect fraudulent financial transactions.
- Compare different machine learning models.
- Build an interactive Power BI dashboard.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Microsoft Power BI

## 🤖 Machine Learning Models
- Isolation Forest
- One-Class SVM
- Random Forest

## 📊 Dashboard Features
- Total Transactions
- Fraud Transactions
- Fraud vs Non-Fraud Distribution
- Transaction Analysis
- Model Performance Comparison
- Fraud Trend Analysis

## 📂 Files Included
- Financial_Transaction.csv
- Financial_Transaction_Anomaly_Detection.ipynb
- Fraud_Detection.pbix
- Project_Report.pdf
